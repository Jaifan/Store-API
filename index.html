Create Read Update & Delete API with NodeJS and MongoDB

This is an Index Page for api please visit : "api/v1/product/"

With "api/v1/product/" -> get & create methods works

with "api/v1/product/:id" get,update & delete methods works

Object Schema is like{ "name": "Tablet", "featured": true, "price": 50, "rating": 4.3, "company": "apple" }


Some Query examle link :

"api/v1/product?name=la" It filters product which name start with "la"

"api/v1/product?numericFilters=rating>4.0,price>=90" It filters product which price grater than or equal 90 And Rating grater than 4.0

"api/v1/product?fields=name,price" It shows one name & price element of each object

"api/v1/product?sort=-price" It sort the list price decending order

"api/v1/product?limit=5&page=3" It shows 11 to 15 obejct list . Because Each page contain 5 element And page No 3.

"api/v1/product?fields=name,price,company&company=vivo&sort=-price&limit=2&page=2" Multiple Query at a same time

"api/v1/product/62dae43bb5b7d7167b83c48f" for single object select where you can perform delete & update with request.body by postman
